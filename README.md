# Твой VPS

Верстка интерфейса сервиса «Твой VPS» — Tailwind CSS 4, шрифт Manrope, адаптивная верстка.

## Страницы

### Авторизация

- [Вход (email)](https://musme.github.io/artem/) — ввод email для получения кода
- [Ввод кода](https://musme.github.io/artem/auth/code.html) — 4-значный код подтверждения
- [Неверный код](https://musme.github.io/artem/auth/code-wrong.html) — состояние ошибки при вводе кода
- [Повторная отправка](https://musme.github.io/artem/auth/code-retry.html) — повторная отправка кода
- [Изменить email](https://musme.github.io/artem/auth/change-email.html) — форма с возможностью изменить email

### Личный кабинет

- [Главная](https://musme.github.io/artem/pages/home.html) — тариф, инструкция, история платежей, поддержка
- [Выбор тарифа](https://musme.github.io/artem/pages/select-plan.html) — карточки тарифов с анимацией при наведении
- [История платежей](https://musme.github.io/artem/pages/payments.html) — полный список платежей
- [Профиль](https://musme.github.io/artem/pages/profile.html) — боковая панель редактирования профиля

### Мобильная версия
